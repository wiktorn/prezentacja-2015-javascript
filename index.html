Składania JavaScript - zmienne

  • Zmienne definujemy poprzezvar nazwa_zmiennej;
  • Typy zmiennych:
    • liczby: 10, 1443.31
    • tekst: "Janek", 'Franek'
    • listy: [1, 2, 4]
    • obiekty/tablice: {imie: "Jan"; nazwisko: "Nowak"}

Składania JavaScript

  • Operatory:
    • +, -, *, /, %, ++, --
    • +=, -=, *=, /=, %=
    • ==, ===, !=, !==, >, <, >=, <=
  • Definicja funkcji: function nazwa(parametr1, parametr2, ...) { //kod return wartość; }

Składania JavaScript

  • Dostęp do własności obiektu: objectName.propertyName, objectName[propertyName], objectName["propertyName"]
  • Powołowyanie nowego obiektu: new NazwaObiektu()

Składania JavaScript

  • Konstrukcje warunkowe: if (warunek1) { // kod } else if (warunek2) { // kod } else { //kod }
  • Pętle: for(inicjalizacja; warunek; wyrażenie wykonywane na koniec pętli) { }
    for(i = 0; i < 10; i++) { }
  • jQuery: $.each(['elem1', 'elem2', 'elem3'], function (index, obj) { // kod });

Składania JavaScript

  • Operacje na listach:
    • [1, 2, 3].push(elem1) === [1, 2, 3, elem1]
    • [1, 2, 3].pop() === 3 a z listy 3 znika
    • [1, 2, 3].shift() === 1 a z listy 1 znika
    • [1, 2, 3].reverse() === [3, 2, 1]
    • [1, 2, 3].slice(1, 3) === [2, 3]
    • [2, 1, 3].sort() === [1, 2, 3] uwaga - inplace
    • [1, 2, 3].splice(0, 2, [4, 5,]) === [3, 4, 5]
    • [1, 2, 3].unshift(-1) === [-1, 1, 2, 3]

Debugowanie JavaScript

  • console.assert(warunek, opis)
  • console.count(opis)
  • console.log(tekst)
  • console.log("Tak wyglada obiekt: %O", obiekt)
  • console.dir(obiekt)
  • console.table(tablica, [kolumna_1, kolumna_2 ...,])
  • console.trace()
